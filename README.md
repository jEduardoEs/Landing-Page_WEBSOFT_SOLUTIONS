# WebSoft Solutions — Landing Page

Landing page oficial de **WebSoft Solutions**, empresa guatemalteca especializada en tecnología, accesorios, periféricos, sistemas CCTV y servicios de mantenimiento y reparación.

## 🗂️ Estructura del proyecto

```
websoft-solutions/
├── index.html        # Página principal
├── css/
│   └── style.css     # Estilos (variables, layout, responsive)
├── js/
│   └── main.js       # Interactividad (WhatsApp, menú, animaciones)
├── assets/           # Imágenes, logo, íconos (agregar aquí)
└── README.md
```

## 🚀 Cómo usar

1. Clona o descarga el repositorio.
2. Abre `index.html` en tu navegador — no necesitas servidor.
3. Para cambios en producción, sube los archivos a tu hosting o usa GitHub Pages.

## 📦 Tecnologías

- HTML5 semántico
- CSS3 puro (variables, grid, flexbox, animaciones)
- JavaScript vanilla (sin frameworks)
- Google Fonts: Rajdhani + DM Sans

## ✏️ Personalización rápida

| Qué cambiar | Dónde |
|---|---|
| Número de WhatsApp | `js/main.js` → función `openWA()` |
| Colores principales | `css/style.css` → bloque `:root` |
| Textos y contenido | `index.html` |
| Logo / imágenes | Carpeta `assets/` |

## 🌐 Deploy en GitHub Pages

1. Sube el proyecto a un repositorio de GitHub.
2. Ve a **Settings → Pages**.
3. En *Source*, selecciona la rama `main` y la carpeta `/ (root)`.
4. ¡Listo! Tu sitio estará en `https://tu-usuario.github.io/websoft-solutions/`
